getCharacterApiUrl()

getCharacterApiUrl(string  $character, string  $server, array|string  $fields = null) : string

Helper method for retrieving character info.

To use this simply implement your method of requesting the data and parsing the return (we use guzzle but are keeping the git generic).

Parameters

string $character

The name of the character to retrieve - this may need to be url encoded.
string $server

The name of the server on which the character resides. - this may need to be url encoded.
array|string $fields

The character fields to retrieve, either an array or comma seperated string.
